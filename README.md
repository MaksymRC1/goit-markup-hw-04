Залишилось:
Feature-картки не містять спеціального обгортання для іконок: SVG-елементи не мають блочного контейнера із фоном, рамкою та вирівнюванням іконки по центру, а самі SVG елементи не мають заданих явно атрибутів width="64" і height="64" як вимагають іконки-спрайти.
Соціальні посилання учасників команди не структуровані як <ul> із елементами <li>, тому бракує необхідної структури списку та керування відступами.
Соціальні SVG-іконки команди оголошено розміром 40×40 замість необхідного розміру іконки 16×16, а hover-стан соціальних кнопок ніколи не спрацьовує, оскільки стилі застосовуються до .social-link:hover::after, а не до .social-link:hover.
Соцмедіа-блок footer також пропускає структуру списку <ul>, і колір фону при hover/focus залишається #404bbf замість необхідного #31d0aa.
